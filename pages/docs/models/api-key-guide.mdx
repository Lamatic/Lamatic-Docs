This guide explains how to add API keys for different LLM providers to your Lamatic.ai account.

## Quick Setup Steps for Each Provider

### OpenAI
1. Visit [OpenAI API]
2. Sign in or create an [account][OpenAI signup]
3. Go to API section → Generate new key
4. In [Lamatic.ai dashboard]: Models → OpenAI → Paste key

![OpenAI](/assets/open-ai-api-image.png)
*OpenAI API Key Generation Screen*

### Anthropic
1. Visit [Anthropic console]
2. Access [API Keys section][Anthropic keys]
3. Create new API key
4. In [Lamatic.ai dashboard]: Models → Anthropic → Paste key

![Anthropic](/assets/anthropic-api-image.png)
*Anthropic Console API Key Screen*

### Azure
1. Go to [Azure portal]
2. Visit [Cognitive Services][Azure cognitive]
3. Find Keys and Endpoint section
4. In [Lamatic.ai dashboard]: Models → Azure → Paste key

![Azure](/assets/azure-api-key-image.png)
*Azure OpenAI Service Keys and Endpoint Section*

### Gemini
1. Visit [Google AI Studio][Gemini studio]
2. Access API keys section
3. Generate new key
4. In [Lamatic.ai dashboard]: Models → Gemini → Paste key

![Gemini](/assets/gemini-api-image.png)
*Google AI Studio API Key Generation Screen*

### Amazon Bedrock
1. Log into [AWS console]
2. Navigate to [Bedrock service][AWS Bedrock]
3. Visit [Security credentials][AWS credentials]
4. In [Lamatic.ai dashboard]: Models → Amazon Bedrock → Paste key

![Amazon Bedrock](/assets/amazon-bedrock-image.png)
*AWS Bedrock Security Credentials Screen*

### Mistral AI
1. Visit [Mistral platform]
2. Go to [API Keys section][Mistral keys]
3. Generate new key
4. In [Lamatic.ai dashboard]: Models → Mistral AI → Paste key

![Mistal AI](/assets/mistral-ai-api-image.png)
*Mistral AI Platform API Key Generation Screen*

## Adding Keys to Lamatic.ai

For all providers, follow these general steps in Lamatic.ai:
1. Open your [Lamatic.ai dashboard]
2. Navigate to Models section
3. Select your desired provider
4. Paste the API key in the designated field
5. Save your changes

![Lamatic.ai Integration Screen](/assets/images/integrations/lamatic-dashboard.png)
*Lamatic.ai Model Integration Dashboard*

## Important Notes
- Keep your API keys secure and never share them
- Some providers may require additional setup steps
- Check each provider's pricing before generating API keys:
  - [OpenAI pricing]
  - [Anthropic pricing]
  - [Azure pricing]
  - [Gemini pricing]
  - [AWS pricing]
  - [Mistral pricing]
- Regularly rotate your API keys for security
- Test your integration after adding each key
