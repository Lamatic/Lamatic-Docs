# AI Nodes

AI nodes help you generate, classify, reason, and call tools in your flow.

## Available AI Nodes

| Node | What it does |
| --- | --- |
| [Generate Text](/docs/nodes/ai/generate-text-node) | Generates natural language responses from prompts and context. |
| [Generate JSON](/docs/nodes/ai/generate-json-node) | Produces structured JSON output for downstream automation. |
| [Generate Image](/docs/nodes/ai/generate-image-node) | Generates images from text instructions. |
| [Classify Input](/docs/nodes/ai/classifier-node) | Classifies incoming content into categories or labels. |
| [Multimodal](/docs/nodes/ai/multimodal-node) | Uses text and image inputs in a single model call. |
| [RAG](/docs/nodes/ai/rag-node) | Retrieves context and generates grounded responses. |
| [Supervise Agents](/docs/nodes/ai/supervisor-node) | Orchestrates multi-agent behavior and task routing. |
| [Call MCP Tools](/docs/nodes/ai/mcp-node) | Connects model reasoning with MCP tool execution. |
