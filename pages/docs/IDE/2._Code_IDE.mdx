## **Code IDE**

Unlock the power of customization with the **Code IDE** in Lamatic workflows. This feature allows you to write concise scripts or execute any JavaScript code directly within your workflow. Whether you need to transform input data, manipulate output data, or perform preprocessing tasks, the Code IDE provides the flexibility to tailor your workflow.

### Key capabilities:
- Identify and merge duplicate records with custom logic
- Iterate over array objects and apply custom mappings or transformations
- Fetch external API data (e.g., currency conversion rates)
- Implement complex business logic and calculations
- Integrate with external libraries and services to extend functionality
