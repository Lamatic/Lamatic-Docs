# Deepinfra 
DeepInfra is a cloud infrastructure platform focused on making AI model deployment and serving more accessible and cost-effective.
<br/>
1. Visit [deepinfra Dashboard][deepinfra Dashboard]
2. Access API keys section
3. Generate new key
4. In [Lamatic.ai Studio]: Models → deepinfra → Paste key

![deepinfra](./img/deepinfra.png)

### Follow these general steps in Lamatic.ai:
1. Open your [Lamatic.ai Studio](https://studio.lamatic.ai)
2. Navigate to Models section
3. Select Deepinfra provider
4. Paste the API key in the designated field
5. Save your changes

## Important Notes
- Keep your API keys secure and never share them
- Some providers may require additional setup steps
- Regularly rotate your API keys for security
- Test your integration after adding each key
