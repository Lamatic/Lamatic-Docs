# Quickstart Guide

Welcome to Lamatic.ai! This guide will take you from a brand-new account to your first working AI Flow in under 10 minutes. 

## 📋 What You'll Need (Prerequisites)
Before we begin, ensure you have:
1. **A Lamatic.ai Account:** [Sign up here](https://lamatic.ai/) if you haven't already.
2. **Basic API Knowledge:** Helpful, but not strictly required for this visual builder.

---

## Step-by-Step Guide

### Step 1: Create Your First Project
Your project is the workspace where your AI Agents and Flows live.
1. Log into the Lamatic.ai Studio.
2. Click the **+ New Project** button on your dashboard.
3. Give your project a recognizable name (e.g., "Customer Support Bot") and click **Create**.

*(Insert Screenshot: New Project Modal)*

### Step 2: Choose a Template or Start Blank
Templates are pre-built configurations designed for specific use cases.
1. Navigate to the **Flows** tab on the left sidebar.
2. Click **Create Flow**.
3. For this guide, select the **"Basic Chat Assistant"** template to get a head start. 

*(Insert Screenshot: Template Selection Screen)*

### Step 3: Configure Your AI Node
Nodes are the building blocks of your Flow. The LLM Node is the "brain."
1. Click on the **Generate Text (LLM)** node in your visual editor.
2. In the right-hand properties panel, select your preferred model (e.g., GPT-4o or Claude 3).
3. Review the default system prompt and adjust it if necessary.

*(Insert Screenshot: Node Configuration Panel)*

### Step 4: Test Your Flow in the Studio
Always test your logic before deploying it to production.
1. Click the **Test** or **Run** button in the top right corner.
2. Enter a sample input in the chat interface (e.g., "Hello, what can you do?").
3. Verify that the AI responds correctly based on your configuration.

### Step 5: Get Your API Keys
To connect this Flow to your actual app or website, you need your keys.
1. Go to **Settings > API Keys**.
2. Click **Generate New Key**.
3. **Important:** Copy this key immediately and store it securely. You will not be able to see it again!

*(Insert Screenshot: API Key Generation)*

## 🛑 Troubleshooting Common Mistakes

* **"Invalid API Key" Error:** Double-check that you copied the entire key and haven't accidentally included any spaces at the beginning or end.
* **Flow isn't responding:** Ensure all your nodes are properly connected in the visual editor. An unconnected node will break the chain.
* **Model access denied:** Make sure your billing is set up if you are using premium models that aren't included in the free tier.
