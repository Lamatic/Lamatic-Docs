# Data Nodes

Data nodes help you ingest, transform, search, and retrieve context.

## Available Data Nodes

| Node | What it does |
| --- | --- |
| [Call an API](/docs/nodes/data/api-node) | Sends HTTP requests to external APIs and services. |
| [Vectorize Content](/docs/nodes/data/vectorize-node) | Converts text or files into vector embeddings. |
| [Chunk Documents](/docs/nodes/data/chunking-node) | Splits large content into smaller retrieval-ready chunks. |
| [Query VectorDB](/docs/nodes/data/vectordb-node) | Retrieves relevant documents from your vector database. |
| [Extract from File](/docs/nodes/data/extract-from-file-node) | Parses and extracts text/content from uploaded files. |
| [Hybrid Search](/docs/nodes/data/hybrid-search-node) | Combines keyword and vector retrieval for better recall. |
| [Keyword Search](/docs/nodes/data/keyword-search-node) | Finds matches using term-based search. |
| [Vector Search](/docs/nodes/data/vector-search-node) | Finds semantically similar content using embeddings. |
| [Add to Memory](/docs/nodes/data/memory-add-node) | Stores useful context into memory for later use. |
| [Retrieve from Memory](/docs/nodes/data/memory-retrieve-node) | Fetches previously stored memory context into the flow. |
