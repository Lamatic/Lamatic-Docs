## Accessing the chosen LLMs
With valid credentials in place, your Lamatic.ai application can now access the LLM via the provider's API. No more worrying about API keys, authentication flows, or managing HTTP requests!
Lamatic.ai abstracts away this complexity with a visual, no-code interface to call the model. Simply select your desired provider and LLM, configure any necessary parameters, and execute the request.
