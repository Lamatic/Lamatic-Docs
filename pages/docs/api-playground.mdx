# API Playground

The **API Playground** is an interactive GraphQL IDE powered by GraphiQL that allows you to explore, test, and optimize your GraphQL queries, mutations, and subscriptions within Lamatic.

## Key Features

- **Interactive Query Builder**: Auto-completion and syntax highlighting for faster query writing
- **Schema Explorer**: Real-time documentation explorer for your GraphQL schema
- **Result Viewer**: Query result viewer with collapsible fields for easy navigation
- **Query History**: History of recent queries for quick reference and reuse

## Getting Started

1. Navigate to **API Playground** in your project
2. Explore your schema in the documentation panel
3. Use auto-completion for faster query writing
4. The IDE validates your syntax as you type
