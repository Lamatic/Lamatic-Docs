### Cloudflare
1. Visit [Cloudflare Dashboard][Cloudflare Dashboard]
1. Navigate to Overview to find Account ID
2. Navigate to Profile to Access API keys
3. Generate new key
4. In [Lamatic.ai dashboard]: Models → Cloudflare → Paste key

![Cloudflare](./img/Cloudflare.png)

Follow these general steps in Lamatic.ai:
1. Open your [Lamatic.ai dashboard]
2. Navigate to Models section
3. Select your desired provider
4. Paste the API key in the designated field
5. Save your changes

## Important Notes
- Keep your API keys secure and never share them
- Some providers may require additional setup steps
- Regularly rotate your API keys for security
- Test your integration after adding each key
