## Adding API Keys for LLM Models

To integrate various Large Language Models (LLMs) with Lamatic.ai, you'll need to add their respective API keys. Follow the steps below to add API keys from different LLM providers:

### 1. OpenAI

1. **Sign Up / Log In**: Go to the [OpenAI website](https://openai.com/index/openai-api/) and sign up or log in to your account.
2. **Navigate to API Keys**: In your account dashboard, locate the API keys section.
3. **Generate API Key**: Click on the button to generate a new API key. Copy the key.
4. **Add to Lamatic.ai**: In the Lamatic.ai dashboard, navigate to the Models section and select "OpenAI". Paste the copied API key into the provided field.

![OpenAI](/assets/open-ai-api-image.png)


### 2. Anthropic

1. **Sign Up / Log In**: Visit the [Anthropic website](https://www.anthropic.com/api) and create an account or log in.
2. **Access API Keys**: Go to your account settings and find the API keys section.
3. **Generate API Key**: Generate a new API key and copy it.
4. **Add to Lamatic.ai**: In the Lamatic.ai dashboard, navigate to the Models section and select "Anthropic". Paste the copied API key into the provided field.

![Anthropic](/assets/anthropic-api-image.png)


### 3. Azure

1. **Sign Up / Log In**: Go to the [Azure portal](https://portal.azure.com/) and log in to your account.
2. **Create Cognitive Services Resource**: In the Azure portal, navigate to the Cognitive Services section and create a new resource if you haven't already.
3. **Retrieve API Key**: Once the resource is created, navigate to the Keys and Endpoint section to get your API key.
4. **Add to Lamatic.ai**: In the Lamatic.ai dashboard, navigate to the Models section and select "Azure". Paste the copied API key into the provided field.

![Azure](/assets/azure-api-key-image.png)


### 4. Gemini

1. **Sign Up / Log In**: Visit the [Gemini AI website](https://ai.google.dev/gemini-api/docs/api-key) and sign up or log in.
2. **Navigate to API Keys**: In your account dashboard, go to the API keys section.
3. **Generate API Key**: Generate a new API key and copy it.
4. **Add to Lamatic.ai**: In the Lamatic.ai dashboard, navigate to the Models section and select "Gemini". Paste the copied API key into the provided field.

![Gemini](/assets/gemini-api-image.png)


### 5. Amazon Bedrock

1. **Sign Up / Log In**: Go to the [Amazon Web Services (AWS) Management Console](https://aws.amazon.com/bedrock/) and log in.
2. **Access Bedrock**: Navigate to the Amazon Bedrock service.
3. **Retrieve API Key**: Find the API key section within the Bedrock service settings.
4. **Add to Lamatic.ai**: In the Lamatic.ai dashboard, navigate to the Models section and select "Amazon Bedrock". Paste the copied API key into the provided field.

![Amazon Bedrock](/assets/amazon-bedrock-image.png)


### 6. Mistral AI

1. **Sign Up / Log In**: Visit the [Mistral AI website](https://auth.mistral.ai/ui/login) and create an account or log in.
2. **Navigate to API Keys**: Go to your account dashboard and find the API keys section.
3. **Generate API Key**: Generate a new API key and copy it.
4. **Add to Lamatic.ai**: In the Lamatic.ai dashboard, navigate to the Models section and select "Mistral AI". Paste the copied API key into the provided field.

![Mistal AI](/assets/mistral-ai-api-image.png)



**By following these steps, you can seamlessly integrate the respective LLM models into Lamatic.ai, enabling enhanced functionality and access to a variety of language processing capabilities.**
